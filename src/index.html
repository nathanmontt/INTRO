45. DOM

Conteúdo estudado do canal: Bro Code

  • → Aqui entra alguns conceitos sobre DOM (Document Object Model);
  • → Sobre os seletores de elementos, dizemos que são métodos usados para encontrar e manipular elementos HTML, podendo ser um ou vários;

Para testá-los, usamos 5 tipos:

  • → document.getElementById(): retorna um ELEMENT ou NULL;
  • → document.getElementsClassName(): retorna uma HTML COLLECTION;
  • → document.getElementsByTagName(): retorna uma HTML COLLECTION;
  • → document.querySelector(): retorna o FIRST ELEMENT ou NULL;
  • → document.querySelectorAll(): retorna uma NODELIST;
  • → Passando um pouco mais para frente, temos as classList, que é uma propriedade usada para interagir com uma lista de classes de um elemento (normalmente CSS);

Temos algumas opções (usaremos um button para teste):

  • → .add(): adiciona classe(s) a um determinado elemento;
  • → .remove(): remove classe(s) de um determinado elemento;
  • → .toggle(): funciona como se fosse um interruptor, "adicionando e removendo" classe(s);
  • → .contains('className'): verifica se a className está contida dentro daquele elemento;
  • → .replace('oldClass', 'newClass'): troca a(s) classe(s) de um determinado elemento;

47. Promisses

Conteúdo estudado do canal: Bro Code

  • → É um objeto que administra asynchronous operations. Operações que podem tomar um certo tempo, e que não queremos que elas pausem todo o resto dos outros códigos;
  • → A ideia é "prometer retornar um valor", sendo esse valor poder ser aceito ou rejeitado (isso enquanto a tarefa fica em pending, espera);
  • → Para escrever: new Promisse((resolve, reject) => {asynchronous code});

48. Async/Await

Conteúdo estudado do canal: Bro Code

  • → Async: faz uma função retornar uma Promisse;
  • → Await: faz uma função async esperar pela em>Promisse;
  • → Isso permite que escrevamos códigos assíncronos de maneira síncrona;

49. fetch()

Conteúdo estudado do canal: Bro Code

  • → É uma função usada para fazer HTTP requests para fetch resources;
  • → Escreve-se da seguinte forma: fetch(url, {options});
